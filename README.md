# stardust-homelab
This is an ongoing IT home lab project where I will build a virtualized environment using Ubuntu Linux, Docker, and industry‑standard tools to grow my skills in system administration, secure configuration, and infrastructure fundamentals.

## Project Goals:
-Learn virtualization using Oracle VirtualBox
-Install and configure a Linux VM (Ubuntu 22.04)
-Practice IT system administration (updates, user management, packages)
-Set up Docker and run containerized applications
-Explore basic networking and service deployment
-Document progress for professional development and transparency

## Why This Project?

This home lab will allow me to build real, hands‑on IT experience in a safe environment. It's designed to grow over time as I develop stronger technical skills relevant to enterprise IT and cybersecurity roles.
